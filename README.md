# Beginner Finance Hub

A calm, practical personal finance site for beginners.

Built with [Astro](https://astro.build), version-controlled on GitHub, auto-deployed via Netlify.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Netlify auto-deploys on push to main. Do not push to production without founder approval.

## Structure

- `src/pages/` - all site pages
- `src/content/articles/` - markdown articles (add here to publish)
- `src/layouts/` - page layouts
- `src/components/` - reusable components
